# J Maslanka Estates – SMS Alert System

A Python-based SMS opt-in system using Twilio, Cloudflare Tunnel, and SQLite.

## Features
- QR-based SMS opt-in
- Name capture
- Auto opt-out (STOP)
- A2P-compliant flow
- Bulk SMS support

## Setup
1. Clone repo
2. Create `.env` file
3. Install dependencies
4. Run webhook

⚠️ Secrets are not included.
